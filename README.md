# lan-video-player
Video player with LAN controls.


## Controls
#### List of Contol Options
- Play/Pause
- Progress Bar (Seeking)
- Volume Controls (Mute/Volume bar)
- Full Screen Toggle
- Forward/Backwards (10s)
- Playback Rates
-

## Tech
- Video player: video.js
- Desktop app: Electron
- Server: express
- Client-server comms: socket.io
